## Run Locally

Clone the project

```bash
  git clone https://github.com/Makita7/ecommerce-makita2.git
```

Go to the project directory

```bash
  cd ecommerce-makita2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
